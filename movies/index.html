小偷家族 / 万引き家族
2019-10-26 / 人性 社会 剧情 日本 家庭 / ★★★★★ 力荐
上映的时候就很有人气。但是直到Amazon Prime可以看的时候才去看。 演员的演技超一流。剧情也超级好。 个人觉得日本底层那些人的生活的一个缩影。 不过最后结局和大部分日本电影一样。残缺的但美好的。
亿男 / 億男
2019-10-26 / 人生 成长 人性 金钱 社会 / ★★★☆☆ 还行
昨晚TBS播出这部电影。说实话，刚开始看了一会就想直接关掉了。后面看了下原版小说的评价，挺高。觉得电影不会那么肤浅和老套的，所以就坚持看了下去。 看完的感觉是，可以看，但不是说很吸引人。
公主大对换 / The Princess Switch
2018-11-18 / 喜剧 爱情 圣诞 / ★★★☆☆ 还行
看了预告片，有点搞笑，感觉音乐也不错。然后就打开看了。虽然有些地方还是挺搞笑的。但是整体还是平平淡淡。
道歉大师 / 謝罪の王様
2018-10-23 / 喜剧 日本 阿部サダヲ / ★★★★☆ 推荐
各种小故事串联在一起，感觉编剧好厉害。隔三差五就让人大笑，但是还是没到让自己笑倒在地那种程度。 整个电影很多小细节都非常棒，也让人感觉很好衬托起整部电影的氛围。
致所有我曾爱过的男孩 / To All the Boys I've Loved Before
2018-10-12 / 人生 成长 校园 青春 喜剧 / ★★★★☆ 推荐
Twitter上评论比较多的一部电影，主要是议论为什么亚裔少女的喜欢的对象里面全部都是非亚裔。一度被认为有种族歧视。 Anyway，整部片子情节不错，同时里面的音乐也不错。
圣之青春 / 聖の青春
2018-10-10 / 日本 传记 人生 / ★★★☆☆ 还行
先喜欢上片尾曲，然后才知道的这部电影。 影片有类似日本那种物哀那种氛围。 圣的青春，正如努力绽放然后凋零的花一样。 故事静静地讲述，如果没有共感到里面的思想的话，可以会觉得无聊而昏昏入睡。
传奇的诞生 / Pelé: Birth of a Legend
2018-10-10 / 运动 体育 足球 励志 成长 / ★★★★★ 力荐
个人觉得非常不错。故事情节也很好。不得不说的是，这电影的BGM是一个超大亮点。
青空呐喊 / 青空エール
2018-10-09 / 校园 日本 青春 / ★★★☆☆ 还行
日式青春电影。整体感觉马马虎虎。很多细节处理地不好。一大堆日式跑。
